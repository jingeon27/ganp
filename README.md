# 저의 개인프로젝트(client) Repository입니다

## 프로젝트 목표

---

저는 개인프로젝트로 여러 **서비스**를 구축해 _플랫폼화_ 시키는 게 목표입니다.

---

## 프로젝트 구성

---

기본적으로 yarn workspace를 이용하여 monorepo를 구성하였습니다. 패키지 관리는 yarn berry를 사용하여 관리하였습니다. react-native의 경우 pnp기능을 활용하여 node_modules로 관리합니다.
